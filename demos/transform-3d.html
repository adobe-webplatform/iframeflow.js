Flowing content into transformed CSS Regions

Tip: resize the browser window to see content flowing from one region to another.

Any CSS properties that apply to regular elements apply to regions as well.

This example shows how content flows from one region to another and respects the rotation transformation applied.

Sample

#region2{
flow-from: article;
transform: rotateY(25deg);
}

Content renders rotated according to the transforms on #region2

Regions don't have to be regular HTML elements. Upcoming implementations will illustrate how CSS pseudo-elements such as ::before or ::after and other generated blocks may be turned into regions.
